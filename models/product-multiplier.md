# 功能-外观-叙事交叉乘数模型

## 核心公式

### 1. 复合增长公式

```
产品势能 = 功能价值 × e^(设计系数 + 叙事系数)
```

- 参数范围
  - 功能价值：0-10
  - 设计系数：0-2
  - 叙事系数：0-2
- 为什么是指数
  - 非线性放大
  - 网络效应
  - 解释爆发现象

### 2. 病毒系数

```
K = i × c × r
```

- i = 邀请数（叙事驱动）
- c = 转化率（外观驱动）
- r = 留存率（功能驱动）
- 临界点：K > 1 自增长

### 3. 转化漏斗

```
最终转化率 = 认知率 × 兴趣率 × 试用率 × 购买率 × 留存率
```

- 叙事 → 认知 × 兴趣
- 外观 → 试用 × 购买
- 功能 → 留存

### 4. 幂律感知

```
感知价值 = 实际价值 × (设计质量)^2 × (叙事强度)^1.5
```

---

## 三个维度

- 功能 (Function)
  - 解决问题的能力
  - 临界点：比现有方案好 3 倍
- 外观 (Form)
  - 视觉设计 + 交互体验
  - 临界点：不能成为使用障碍
- 叙事 (Narrative)
  - 用户如何理解和传播
  - 临界点：能否向朋友推荐

## 案例

- iPhone (2007)
  - 功能：9/10
  - 外观：10/10
  - 叙事：10/10 ("重新发明手机")
  - 势能 = 491
- Tesla Model S
  - 功能：8/10
  - 外观：9/10
  - 叙事：9/10
  - 势能 = 293
- Google Glass (失败)
  - 功能：6/10
  - 外观：3/10
  - 叙事：7/10
  - 势能 = 44
  - 原因：外观拖累

## 应用

- 诊断
  - 功能：留存率、NPS
  - 外观：首次完成率、UI满意度
  - 叙事：推荐率、媒体提及
- 优先级
  - < 5分 → 修复短板
  - > 7分 → 提升强项
- 资源分配
  - 早期：功能 60% + 外观 30% + 叙事 10%
  - 成长期：功能 40% + 外观 30% + 叙事 30%
  - 成熟期：功能 30% + 外观 20% + 叙事 50%

---

## 局限性

- 不是精确科学 - 系数难以量化
- 时间维度缺失 - 模型是静态的
- 市场时机未考虑 - 时机影响结果
- 竞争环境简化 - 需考虑竞品
